Woldwide Weather Watcher

Conception et développement d'une station météo portable.

L'objectif de ce projet était d'assembler et de programmer une petite station météo fonctionnant sur la base d'une carte Arduino et qui récupère périodiquement des informations tels que la température, l'humidité ou la positions GPS et les enregisre sur une carte SD.

Jeu de la Vie en Poo

Simulation d'un jeu de la vie en programmation orientée objet.

Ce projet consistait à simuler le jeu de la vie de John Conway en utilisant la programmation orientée objet. Le jeu de la vie est un automate cellulaire où chaque cellule peut être vivante ou morte, et son état évolue en fonction de règles simples basées sur les états de ses voisins. Nous avons fait ce projet en utilisant le langage python et avons créé une interface graphique pour visualiser l'évolution du jeu. De plus nous avons implémenté un moyen de sauvegarder et de charger des configurations de cellules pour pouvoir les réutiliser ultérieurement. Finalement, nous avont ajouté des paramètres personnalisables pour les règles du jeu, permettant ainsi d'explorer différentes variantes du jeu de la vie.


Ce projet nous a permis de mettre en pratique nos compétences en programmation orientée objet, en développement d'interfaces graphiques et en gestion de fichiers pour créer une simulation interactive du jeu de la vie.

Technologies : Python

Compétences : programmation orientée objet, développement d'interfaces graphiques, gestion de fichiers

Projet Fast and Furious

Simulation du comportement de voitures en Python dans un environnement contrôlé.

Ce projet consistait à simuler le comportement de voitures dans un circuit défini en utilisant Python. Pour cela nous avons utilisé la bibliothèque numpy pour les calculs numériques. Ce projet ne comprenait pas d'interface graphique. Mais cette image reporésente le parcours que devaient parcourir les voitures dans le projet.

Nous avions une sélection de voitures avec des caractéristiques différentes (masse, accélération, dimentions, etc.) auxquelles nous pouvions ajouter du nos, une jupe, et/ou ub aileron. Nous devions simuler leur comportement en fonction de ces caractéristiques et du circuit. Nous avons utilisé des équations de mouvement pour calculer la position et la vitesse des voitures à des endroits spécifiques du circuit, en tenant compte de facteurs tels que la friction et les forces appliquées. Le projet nous a permis de mettre en pratique nos compétences en programmation et en physique pour créer une simulation du comportement des voitures sur le circuit.

Technologies : Python

Compétences : simulation, programmation, calcul numérique

Projet Traitement de Signal

Développement d'un système permettant de transmettre un fichier en utilisant un signal simulé.

Dans ce projet de traitement du signal, nous avons conçu un système capable de transformer un fichier numérique en signal transmissible, puis de reconstruire le fichier à partir du signal reçu. Le programme encode les données sous forme de signal, simule leur transmission et applique un processus de décodage pour récupérer les informations d’origine.

Ce projet permet de comprendre les principes fondamentaux de la communication numérique.

Technologies : Python

Compétences : traitement de signal, programmation, encodage et décodage de données, manipulation de fichiers, simulation de transmission

Projet Base de données d'entreprise - SQL

Conception et développement d'une base de données relationnelle permettant de stocker des données de qualité de l'air.

Ce projet consistait à développer une base de données pour centraliser les données publiques relatives à la qualité de l'air dans plusieurs villes de France. Le système devait permettre de gérer les agences régionales responsables des capteurs, le personnel travaillant dans ces agences ainsi que les capteurs mesurant la concentration de différents gaz dans l'atmosphère.

Technologies : SQL, MySQL, Looping MCD

Compétences : modélisation de base de données relationnelles, conception de schémas de données, écriture de requêtes SQL
